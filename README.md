# Node.js Simple Web Server (Laundry Service Website)

This assignment is a simple Node.js web server built using the native `http` module.
It demonstrates manual routing (Home, About, Contact) without using Express and serves dynamic HTML with inline css.

## Features

- Built using pure Node.js (`http` module)
- Manual routing:
  - `/` or `/home`
  - `/about`
  - `/contact`
  - 404 page handling
- HTML generated using a reusable function
- No external frameworks needed
